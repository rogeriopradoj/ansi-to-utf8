# AnsiToUtf8

Totally based on https://github.com/dongri/convert-to-utf8.

AnsiToUTF8 package for editing and saving files encoded in ANSI (Western) Shift_JIS, EUC-JP, CP932, GBK, BIG5, BIG5-HKSCS, EUC-KR etc.
